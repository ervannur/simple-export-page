== Description ==

Export one or more page using bulk action, so it is easy to use. The pages will imported as xml format, hence it is compatible with default Wordpress Importer plugin.

== Installation ==

1. Upload the plugin files to the */wp-content/plugins/simple-export-page* directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the *Plugins* screen in WordPress
3. No setting needed
